## 📋 개요

### 서비스 구조

PPT Maker는 3단계 AI 파이프라인을 통해 프리젠테이션을 생성합니다:

```
사용자 입력
  → 1️⃣ 자료 조사 (Perplexity - 선택)
  → 2️⃣ 콘텐츠+JSON 생성 (Gemini Flash/Pro - Parser 통합)
  → 3️⃣ 템플릿 렌더링 (클라이언트 엔진 - 0원)
```

**핵심 개선점**:
- ✅ **Parser 통합**: 콘텐츠 생성 시 UnifiedPPTJSON 형식 직접 생성 (8원 절감)
- ✅ **HTML 클라이언트 처리**: 템플릿 엔진으로 비용 100% 절감 (75-82원 절감)

### 수익 모델 개요

**하이브리드 수익 모델** (5가지 수익 채널):

- **무료 사용자**: 광고 시청 (전면광고 2회) + 기본 품질 무제한 + 최초 1회 무료 (심층 검색 + 고품질 생성)
- **구독 사용자**: Pro 플랜 (₩4,900/월) → 광고 제거 + 매월 490 크래딧 제공 + 워터마크 제거 + 20페이지 제한
- **건당 과금**: 심층 검색 40 크래딧 (₩400), 고품질 생성 50 크래딧 (₩500)
- **묶음 구매**: 크레딧 패키지 (100/300/500/1000 크래딧)
- **IAP**: 프리미엄 템플릿 (₩900~₩4,900)

## 💰 AI 원가 분석

### 단계별 AI 모델 구성

#### 1단계 (선택): 자료 조사 (Perplexity AI)

**옵션 1: 자료 조사 안함**
- 비용: **₩0**

**옵션 2: 빠른 조사 (Sonar)**
- 웹 검색을 통한 빠른 자료 수집
- 평균 토큰 사용량: 입력 1,000 + 출력 2,000 = **3,000 tokens**
- 단가: $1.00 / 1M tokens (입력/출력 동일)

**옵션 3: 깊은 조사 (Sonar Reasoning)**
- 논리적 추론과 분석
- 평균 토큰 사용량: 입력 2,000 + 출력 4,000 = **6,000 tokens**
- 단가: 입력 $1.00, 출력 $5.00 / 1M tokens

#### 2단계: 콘텐츠+JSON 생성 (Gemini)

**옵션 1: Gemini 2.5 Flash (빠른 속도)**
- 평균 토큰 사용량: 입력 3,000 + 출력 2,000 = **5,000 tokens**
- UnifiedPPTJSON 형식 직접 생성 (Parser 단계 제거)
- 단가: 입력 $0.30, 출력 $2.50 / 1M tokens

**옵션 2: Gemini 2.5 Pro (고품질) - 실제 측정값**
- 실제 토큰 사용량: 입력 9,000 + 출력 2,000 = **11,000 tokens**
- 3,330자 콘텐츠 + UnifiedPPTJSON 형식 생성 (시각화 포함)
- 단가: 입력 $1.25, 출력 $10.00 / 1M tokens

- **비용: ₩0** (클라이언트 측 처리)
