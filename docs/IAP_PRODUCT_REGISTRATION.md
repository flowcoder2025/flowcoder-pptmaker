## ⚠️ 등록 후 필수 작업

### 1️⃣ 코드에 실제 SKU 반영

콘솔에서 등록 완료 후, `services/iap/config.ts` 파일을 업데이트하세요:

```typescript
export const PRODUCT_SKUS = {
  subscription: {
    standard: 'pptmaker_standard_monthly',  // ✅ 콘솔에서 확인
    premium: 'pptmaker_premium_monthly',    // ✅ 콘솔에서 확인
  },
  credits: {
    small: 'pptmaker_credits_10',    // ✅ 콘솔에서 확인
    medium: 'pptmaker_credits_50',   // ✅ 콘솔에서 확인
    large: 'pptmaker_credits_100',   // ✅ 콘솔에서 확인
  },
} as const;
```

### 2️⃣ 샌드박스 테스트

- Apps in Toss 앱에서 샌드박스 모드로 전환
- 각 상품별 결제 테스트 진행
- 상품 지급 로직 확인

### 3️⃣ 프로덕션 배포

- 모든 테스트 통과 후 프로덕션 배포
- 실제 결제 테스트 (소액)
